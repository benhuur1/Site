Lista de plugins que utilizo no Sublime Text 3

Lista com 19 Plugins que utilizo no Sublime Text 3

1- Emmet

  • Digita uma forma simplificado para criação de códigos tags HTML depois aberta tag e gera códigos mais rápidamente.

    • 2- auto-save

  • Habiliza o salvamento automático através de um atalho no teclado

    • Para Habilita-lo basta ir até Preferences > Key Brindinds e colocar e seguinte código: { "keys": ["f10"], "command": "auto_save" }



    Ele necessita ser habilita toda vez que for utilizar no meu caso estou utizando o atalho F12

    3- Auto indentar os códigos

  • Através de um atalho de teclado você indenta os códigos
    • { "keys": ["f12"], "command": "reindent", "args": {"single_line": false}}

    Para a utilização desse comando não é necessário instalar plugins



    4- AutoFileName

  • Mostra o nome de pastas e arquivos.

    • 4- BracketHighlighter

  • Mostra a abertura e fechamento de tags, chaves e colchetes nos códigos

    • 5- LoremIpsum

  • Gera text Lorem Ipsum automáticamente lorem*4 depois aperta a tecla Tab e gerara o texto com 4 palavras.

    • 6- Color Highlighter

  • Mostra como a cor no código vai ficar na aplicação

    • 7- ColorPicker

  • Abre uma paleta de cores para escolher ao abertar o atalho Ctrl+Shift+C

    • 8- HTML5

  • Coloeçã de tags HTML5 para auto completar o código

    • 9- CSS Extended Completions

  • Um auto completavel de códigos CSS

    • 10- CSS Snippets

  • Como o próprio nome já fala é uma coleção pronta de códigos CSS

    • 11- FileIcons

  • Mostra o icone da extenção do arquivo por exemplo se seu arquivo for extensão .html mostra o icone do HTML

    • 12- JavaScript Completions

  • Auto Completavel para a JavaScript

    • 13- JavaScriptNext - ES6 Syntax

  • Sintaxe Ecma Script 6

    • 14- jQuery

  • Auto Completáveis para funções da biblioteca JQuery

    • 15- Material Theme

  • Mudar o visual do Sublime Text 3

    • 16- PHPIntel

  • Completar automaticamente para funções e objetos PHP

    • 17- SideBarEnhancements

  • Aprimoramentos na barra lateral Sublime Text. Arquivos e pastas.

    • 18- Bootstrap 4 Autocomplete

  • Auto completa com as classes do bootstrap 4

    • 19- AdvancedNewFile

  • Criação Rápida de novos arquivos
